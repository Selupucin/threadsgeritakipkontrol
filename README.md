## 🚀 Kurulum (Chrome / Edge)

> Beta sürüm "geliştirici modu" ile kurulur (Store onayına kadar normaldir).

1. Bu repoyu indir: yeşil **`Code` → `Download ZIP`** → ZIP'i bir klasöre **çıkar**.
   - (veya `git clone https://github.com/Selupucin/threadsgeritakipkontrol`)
2. Chrome'da adres çubuğuna **`chrome://extensions`** yaz (Edge'de `edge://extensions`).
3. Sağ üstten **Geliştirici modu**'nu **aç**.
4. **Paketlenmemiş öğe yükle** (Load unpacked) → çıkardığın **klasörü seç** (içinde `manifest.json` olan klasör).
5. Araç çubuğunda **🐱 TGTK** ikonu görünür. (Görünmüyorsa puzzle 🧩 simgesine tıklayıp sabitle.)

## ▶️ Kullanım

1. **threads.com**'a gir, hesabına giriş yap.
2. Araç çubuğundaki **🐱 TGTK** → **Kontrol Et**.
3. Eklenti otomatik kendi profilini açıp tarar; sonuç sayfadaki **panelde** çıkar.

## 💛 Not: Takip kapısı

Araç tamamen ücretsizdir. Karşılığında tek ricamız: geliştiriciyi **[@jrselupucin](https://www.threads.com/@jrselupucin)** takip etmen. Takip ettikten sonra sonuçlar gösterilir. İstemezsen kapatabilirsin.

## 🔒 Gizlilik

## ℹ️ Bilinen sınır

Threads çok büyük listeleri (yaklaşık 430–450 kişiden sonra) modal içinde yüklemeyi durduruyor. Çok kalabalık hesaplarda sonuç **taranabilen kısma** göre hesaplanır; eklenti bunu panelde **şeffafça** belirtir. (Bu Threads'in sınırı, eklentinin değil.)

## 🛠️ Sorun giderme

- **İkon yok / çalışmıyor:** `chrome://extensions`'tan eklentiyi **yenile** (↻), sonra Threads sekmesini **F5** ile yenile.
- **Tarama yavaş/eksik:** Çok sık tarama yaptıysan Threads geçici hız sınırı uygulayabilir; birkaç dakika sonra tekrar dene.
